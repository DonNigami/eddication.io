🎉 การติดตั้งหน้าแสดงผล Participant Display

ฟีเจอร์ใหม่: หน้าแสดงผลแบบลอยไปลอยมา

ฟีเจอร์นี้: แสดงรูปโปรไฟล์ LINE ของผู้ที่ check-in แล้ว แบบลอยไปลอยมา ขนาดไม่เท่ากัน เพื่อสร้างความน่าสนใจและตรวจสอบได้ทันทีว่าใครเข้าร่วมกิจกรรมแล้ว

📋 วิธีการใช้งาน

1. เปิดหน้าแสดงผล

เปิดไฟล์ participant-display.html พร้อมระบุ ID ของกิจกรรม: 

participant-display.html?activityId=YOUR_ACTIVITY_ID

2. เพิ่มปุ่มเปิดหน้าแสดงผลในหน้า Check-in

เพิ่มปุ่มใน checkin.html เพื่อให้ admin เปิดดูได้: 

<button onclick="openDisplayPage()">
  📺 เปิดหน้าแสดงผล
</button>

<script>
function openDisplayPage() {
  const activityId = getCurrentActivityId();
  window.open(`participant-display.html?activityId=${activityId}`, '_blank');
}
</script>

3. ตั้งค่า Backend API

อัปเดต URL ใน participant-display.html: 

// แก้ไขบรรทัดประมาณ 458
const response = await fetch('YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'getParticipantsForDisplay',
    activityId: ACTIVITY_ID
  })
});

4. Deploy Google Apps Script

Deploy เป็น Web App และตั้งค่า:

  • Execute as: Me (your email)
  • Who has access: Anyone

🔧 การปรับแต่ง

ปรับขนาดรูปโปรไฟล์

แก้ไข CSS ใน participant-display.html:

.size-small { width: 60px; height: 60px; }
.size-medium { width: 90px; height: 90px; }
.size-large { width: 120px; height: 120px; }
.size-xlarge { width: 150px; height: 150px; }

ปรับสีและ Theme

แก้ไข gradient background:

body {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  /* เปลี่ยนสีตามต้องการ */
}

ปรับความเร็ว Auto-refresh

แก้ไข interval ใน JavaScript:

// เปลี่ยน 30000 เป็นมิลลิวินาทีที่ต้องการ
setInterval(async () => {
  if (IS_AUTO_REFRESH) {
    await loadParticipants();
  }
}, 30000); // 30 วินาที
