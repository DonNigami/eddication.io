⚡ Quick Start: Participant Display

1

Deploy Google Apps Script

  1. เปิด backend/APIHandlers.gs
  2. ไปที่ Deploy → New deployment
  3. เลือก Web app
  4. Execute as: Me (your email)
  5. Who has access: Anyone
  6. กด Deploy
  7. คัดลอก URL ที่ได้
2

อัปเดต participant-display.html

แก้ไขบรรทัดที่ 458:

// แทนที่ YOUR_WEB_APP_URL_HERE ด้วย URL จาก Step 1
const response = await fetch('YOUR_WEB_APP_URL_HERE', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'getParticipantsForDisplay',
    activityId: ACTIVITY_ID
  })
});
3

เปิดหน้าแสดงผล

ใช้ URL นี้:

participant-display.html?activityId=YOUR_ACTIVITY_ID
