🚀 3 ขั้นตอนเริ่มใช้งาน

1

Deploy Google Apps Script

  1. เปิด backend/Code_Old_Reorganized.gs
  2. ไปที่ Deploy → New deployment
  3. เลือก Web app
  4. ⚠️ สำคัญ:
    • • Execute as: Me (your email)
    • • Who has access: Anyone
  5. คลิก Deploy
  6. คัดลอก Web App URL เช่น: https://script.google.com/macros/s/XXXXX/exec
2

อัปเดต participant-display.html

เปิดไฟล์ participant-display.html และแก้ไขบรรทัดที่ 423:

// แทนที่ YOUR_DEPLOYED_WEB_APP_URL_HERE ด้วย URL จาก Step 1
const SCORDS_API_URL = 'https://script.google.com/macros/s/XXXXX/exec';
3

เปิดหน้าแสดงผล

ใช้ URL นี้ (แทนที่ YOUR_ACTIVITY_ID):

participant-display.html?activityId=YOUR_ACTIVITY_ID

💡 หมายเหตุ: Activity ID อยู่ใน Google Sheets → Activities sheet → คอลัมน์ ID

❌ "Failed to load participants"

  • • ตรวจสอบว่า Deploy เรียบร้อย (Execute as: Me, Access: Anyone)
  • • ตรวจสอบว่าอัปเดต SCORDS_API_URL แล้ว
  • • รอ 1-2 นาทีหลัง Deploy แล้วลองใหม่

⚠️ รูปโปรไฟล์ไม่แสดง

  • • ระบบมี fallback เป็น UI Avatar อัตโนมัติ
  • • ตรวจสอบ Users sheet คอลัมน์ ProfilePicture
  • • รูปอาจหมดอายุหรือ URL ไม่ถูกต้อง
