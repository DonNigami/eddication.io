🚀 JETSETGO - Edge Functions Deployment

📋 Deployment Instructions

Due to CLI permission issues, deploy functions manually through Supabase Dashboard:

  1. Go to Supabase Dashboard → Functions
  2. Click "New Function" for each function below
  3. Copy the function code and paste it into the editor
  4. Set environment variables (see below)
  5. Click "Deploy"

🔑 Required Environment Variables

GROQ_API_KEY - Get from groq.com (FREE)
HUGGINGFACE_API_KEY - Optional, from huggingface.co
LINE_CHANNEL_ACCESS_TOKEN - From LINE Developers Console
LINE_CHANNEL_SECRET - From LINE Developers Console
