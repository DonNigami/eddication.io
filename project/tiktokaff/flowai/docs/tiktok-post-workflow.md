## 2. ตั้งเวลาโพส (Schedule Post Flow)

```
┌─────────────────────────────────────────────────────────────────┐
│                    SCHEDULE POST WORKFLOW                       │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────┐           ┌──────────┐           ┌──────────┐
    │  Step 1  │ ────────► │  Step 2  │ ────────► │  Step 3  │
    │ คลิก     │           │ เลือกเวลา │           │ เลือกวันที่│
    │ Schedule │           │ (Time)   │           │ (Date)   │
    └──────────┘           └──────────┘           └──────────┘
                                                       │
                                                       ▼
    ┌──────────┐           ┌──────────┐           ┌──────────┐
    │  Step 6  │ ◄──────── │  Step 5  │ ◄──────── │  Step 4  │
    │ กด Post  │           │ กด Confirm│           │ Show More│
    └──────────┘           └──────────┘           └──────────┘
         │
         ▼
    ┌──────────┐           ┌──────────┐
    │  Step 7  │ ────────► │  Step 8  │ ──────► NEXT CLIP
    │ กด Confirm│           │ กด Upload│
    │ (Popup)  │           │ (กลับหน้า)│
    └──────────┘           └──────────┘
```

### รายละเอียด Steps

| Step | Action | หมายเหตุ |
|------|--------|---------|
| 1 | คลิก **Schedule radio** | เปลี่ยนจาก "Post now" เป็น "Schedule" |
| 2 | เลือก **เวลา** จาก time picker | HH:MM |
| 3 | เลือก **วันที่** จาก calendar | DD/MM/YYYY |
| 4 | คลิก **Show more** | ขยาย options เพิ่มเติม |
| 5 | คลิก **Confirm** ใน options | ยืนยันการตั้งค่า |
| 6 | คลิกปุ่ม **Post/Schedule** | ปุ่มหลักล่างสุด |
| 7 | คลิก **Confirm** บน popup | ยืนยันการโพส |
| 8 | คลิก **Upload** | กลับไปหน้า upload สำหรับคลิปถัดไป |
